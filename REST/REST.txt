* REST (Representational State Transfer)
- REST is an architectural style that defines a set of constraints to br used for creating websites.

* CRUD : Operations
GET retrives resources
POST submits new data 
PUT updates existing data
PATCH update existing partially
DELETE removes data

* CREATING RESTFUL APIs
GET /post : to get data for all posts
POST /posts : to add a new post
GET /posts:id : to get one post (using id)
PATCH  /posts/:id : to update specific post
DELETE /posts/:id : to delete specific post
